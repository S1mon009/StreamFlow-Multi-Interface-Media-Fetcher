# YouTube-downloader (CLI)

## Description
The YouTube-downloader (CLI) project allows you to download movies, playlist and music from YouTube. He uses the **yt_dlp** library. Decorators, auxiliary functions and object -oriented programming were used for concise and clean code.

## Functions
- Downloading individual films and playlist
- Possibility to choose video quality  
- Support for various formats (MP4, MKV) 
- Automatic service of disconnected internet

## Requirements
- Python **3.10+**
- `ffmpeg` (for file conversion)
